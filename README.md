# NestJS SMTP Email Service — Trio Chat Assignment
 
## Quick start

1. Install dependencies:
   ```
   npm install
   ```

2. Create `.env` past values from your credentials

3. Start MongoDB and an SMTP testing server or mongo atlas

4. Run server:
   ```
   npm run start:dev
   ```

## Endpoints

- `POST /emails/send` — send an email
- `GET /emails` — list email logs (supports `page`, `limit`, `status`)
